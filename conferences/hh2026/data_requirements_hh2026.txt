# Format (simple, line-oriented; '#' starts a comment, blank lines ignored):

#   - Inside a block, "key: value" lines give metadata. Recognized keys:
#       required     yes | no   (default yes) — whether the file must be present
#       produced_by  which script normally creates it (fetch / process / manual)
#       description  one-line human description
#       manual       how to obtain the file by hand if the downloader can't
#   - Continuation: a "manual"/"description" value may wrap onto indented lines.
#
